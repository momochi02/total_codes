Tạo giao diện web dashboard hiện đại và chuyên nghiệp để xem kết quả chạy Fotal Update.

Dashboard gồm:

1. Thông tin request
- Request ID
- Device
- Update type
- Status
- Created by
- Latest running step

2. Hệ thống execution log
- Nút mở execution logs
- Drawer/panel bên phải
- Timeline các step
- Mỗi step có status:
  completed, running, failed, pending, skipped
- Click step để xem log chi tiết

3. Kết quả testcase
- Search testcase
- Filter theo status
- Danh sách testcase dạng card
- Hiển thị:
  before result
  after result
  compare result
  final status

4. Popup chi tiết testcase
- Before screenshot
- After screenshot
- Diff image
- Compare summary
- Added / Removed / Changed components
- Logs

5. Navigation trong popup
- Previous testcase
- Next testcase
- Hiển thị nút ở ngoài popup bên trái/phải

Yêu cầu UI:
- Phong cách dashboard QA nội bộ hiện đại
- Light theme
- Background sáng
- White card
- Bo góc mềm
- Shadow nhẹ
- Typography chuyên nghiệp
- Dễ nhìn, dễ scan
- Ít màu sắc
- Responsive desktop layout
- Cảm giác giống tool automation testing chuyên nghiệp
